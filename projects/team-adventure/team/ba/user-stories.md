# User Stories — Virtual Office MVP

## User Stories

### US-001
**As a** Team Member, **I want** to enter my display name and choose an avatar style so that I can spawn inside the virtual office as a recognizable player.
**Priority:** Essential
**Effort:** S (1pt)
**Acceptance:** → acceptance-criteria.md

### US-002
**As a** Team Member, **I want** to walk around the office using arrow keys or WASD so that I can navigate to different desks and rooms.
**Priority:** Essential
**Effort:** M (3pt)
**Acceptance:** → acceptance-criteria.md

### US-003
**As a** Team Member, **I want** to toggle my microphone and camera states so that I can control my privacy while working or meeting.
**Priority:** Essential
**Effort:** S (1pt)
**Acceptance:** → acceptance-criteria.md

### US-004
**As a** Team Member, **I want** to change my status (Working, Meeting, Away) so that my colleagues know my current availability.
**Priority:** Essential
**Effort:** S (1pt)
**Acceptance:** → acceptance-criteria.md

### US-005
**As a** Team Member, **I want** to automatically start a video and audio call when I walk near another colleague in the open office space so that we can chat spontaneously.
**Priority:** Essential
**Effort:** L (5pt)
**Acceptance:** → acceptance-criteria.md

### US-006
**As a** Team Member, **I want** to walk into a private meeting room and only talk with people inside that room so that we can have confidential discussions.
**Priority:** Essential
**Effort:** L (5pt)
**Acceptance:** → acceptance-criteria.md

### US-007
**As a** Team Member, **I want** the voice of my colleagues to get quieter as I move away from them so that it mimics real physical acoustics.
**Priority:** Conditional
**Effort:** M (3pt)
**Acceptance:** → acceptance-criteria.md

### US-008
**As a** Team Member, **I want** to edit a shared notepad and see other people's edits in real-time so that we can coordinate notes and goals.
**Priority:** Essential
**Effort:** M (3pt)
**Acceptance:** → acceptance-criteria.md

### US-009
**As a** Team Member, **I want** to walk up to interactive objects like whiteboards or screens and open them with a button click so that I can use shared applications.
**Priority:** Essential
**Effort:** M (3pt)
**Acceptance:** → acceptance-criteria.md
